since everybody calls different chords by slightly different names I'm going to explain 
some of the more complicated looking ones I used here (don't worry they're accually 
really easy and sound cool too)

For C2/B take off the a string 3rd fret and at the same time move the d string
2nd fret up to the a string 2nd fret (you'll hear it in the song)

For Fmaj7/A play the higher sounding F (d string 3rd fret, g string 2nd fret,
b string 1st fret, high e 1st fret) and leave off the high e string (again you'll
hear it in the song)

For Gadd4 it's just G with the b string 1st fret added

Other then that listen to the song and match the chord changes up with it.I'm
